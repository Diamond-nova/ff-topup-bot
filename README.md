# Telegram Bot для пополнения Free Fire

Гибкий бот для приема заказов на пополнение игровой валюты с системой модерации в группе.

## Основные функции
- 📝 **Прием заказов**: Сбор игрового ID, суммы и скриншота чека.
- 👥 **Группа модераторов**: Все заказы приходят в специальную группу с кнопками управления.
- ⚙️ **Статусы**: "Принято", "В процессе", "Отклонено" (с указанием причины).
- 🗄️ **База данных**: Хранение всех заказов в SQLite.

## Настройка
1. Откройте файл `config.py`.
2. Вставьте ваш `BOT_TOKEN` (получите у @BotFather).
3. Укажите `MODERATOR_GROUP_ID`. Чтобы узнать ID группы, добавьте в нее бота @getmyid_bot или напишите любое сообщение боту и посмотрите логи.
4. (Опционально) Измените номер телефона в `PAYMENT_PHONE`.

## Установка и запуск
1. Установите библиотеку `aiogram`:
   ```bash
   pip install aiogram
   ```
2. Запустите бота:
   ```bash
   python bot.py
   ```

## Как сделать бота гибким?
- **Поля заказа**: Если нужно добавить поле (например, "Сервер"), добавьте новое состояние в `states.py` и соответствующий хендлер в `bot.py`.
- **Логика**: Вы можете добавить проверку суммы или интеграцию с платежными API в будущем.
